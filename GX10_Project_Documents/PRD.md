# Product Requirements Document (PRD)

## Status
- Core concept: COMPLETE
- Feature details: PARTIAL

## Product Vision
A controlled agent-coding system where reasoning and execution are separated,
ensuring maximum code quality and long-term maintainability.

## Key Features
- Agent Coding Pipeline (Claude + Local LLM + GX10)
- GX10 Code Brain (execution engine)
- GX10 Vision Brain (vision algorithm quality engine)
- Idle Improvement (continuous model evolution)

## Non-Goals
- Replace human architectural decisions
- Autonomous requirement discovery

## Assumptions
- Internet access available
- Docker-based deployment

- UI Dashboard (TBD)
