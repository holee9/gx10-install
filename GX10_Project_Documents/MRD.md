# Market Requirements Document (MRD)

## Status
- Overall: PARTIALLY COMPLETE

## Background
The project aims to build a high-quality agent coding system with minimal subscription cost,
leveraging local LLMs and a dedicated AI brain server (GX10). Code quality is the top priority.

## Market Problem
- Rising subscription costs for cloud AI coding tools
- Difficulty maintaining long-term code quality
- Lack of controllable, evolvable local AI coding systems

## Target Users
- Senior developers
- Research engineers
- Automated development pipelines (CI/CD, MCP, n8n)

## Key Market Needs
- High code quality over speed
- Low recurring cost
- Deterministic, controllable AI behavior
- Evolvable local intelligence

## Out of Scope
- Consumer IDE features
- Real-time chat UX
